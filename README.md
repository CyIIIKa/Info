# Работа с git
### Скачать проект в папку
```bash
git clone[https url]
```
### Учесть все изменные файлы
```bash
git add .
```
 ### Сохранить коммит
```bash
git commit -m "..."
```
 ### Отправить на сервер
```bash
git push
```
